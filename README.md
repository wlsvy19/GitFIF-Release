# GitFIF Desktop

GitFIF Desktop 실행 파일 다운로드 전용 저장소입니다.

이 저장소에는 소스 코드가 포함되어 있지 않습니다. GitFIF 소스 저장소는 비공개로 관리하며, 이곳에는 사용자가 설치하거나 실행할 수 있는 빌드 산출물만 게시합니다.

## 다운로드

| OS | 파일 |
| --- | --- |
| Windows x64 | [GitFIF-Windows-x64.zip](downloads/GitFIF-Windows-x64.zip) |
| macOS | [GitFIF-macOS.zip](downloads/GitFIF-macOS.zip) |

## 사용 방법

### Windows

1. `GitFIF-Windows-x64.zip`을 다운로드합니다.
2. 압축을 해제합니다.
3. 압축 해제된 폴더 안의 `GitFIF.exe`를 실행합니다.

### macOS

1. `GitFIF-macOS.zip`을 다운로드합니다.
2. 압축을 해제합니다.
3. `GitFIF.app`을 실행합니다.

## 무결성 확인

다운로드 파일의 SHA-256 해시는 [SHA256SUMS.txt](SHA256SUMS.txt)에서 확인할 수 있습니다.

## 안내

- 공개 저장소에는 다운로드용 파일만 유지합니다.
- 소스 코드, 개발 이력, 빌드 스크립트는 비공개 저장소에서 관리합니다.
- Windows 실행은 zip 내부의 전체 폴더 구조가 필요합니다. `GitFIF.exe`만 따로 분리해서 실행하지 마세요.
